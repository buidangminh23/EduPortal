# 🎓 EduPortal

All-in-one school platform with four role-based experiences — **Principal, Teacher, Parent, Student** — covering academics, communication, school operations, AI tools, and student wellbeing.

> The app lives in the **`Web/`** directory.

## Tech
React 19 · Vite 8 · lucide-react · deployed on Vercel.

## Key features
- Role dashboards (Principal / Teacher / Parent / Student)
- AI tools: AI Tutor, Essay Grader, AI Risk Analysis, University Matchmaker
- Communication: class chat, direct chat, bulletin board, EduMeet
- Academics: exams, video lectures, flashcards, grade trends, mock exams
- School ops: attendance, timetable, seating, canteen, bus tracker, library
- Engagement: badges, leaderboard, wellness hub, study groups, VietQR wallet

## Run
```bash
cd Web
npm install
npm run dev
```

`npm run build` → production build (deployed on Vercel).
